# Database LeetCode Solutions 

This repository contains my solutions to **LeetCode Database (SQL) problems**. The goal of this repository is to strengthen my SQL skills by solving problems covering querying, filtering, joins, aggregations, subqueries, window functions, and advanced database concepts.

## Objectives

* Practice SQL regularly.
* Improve problem-solving skills in databases.
* Build a structured collection of SQL solutions.
* Prepare for coding interviews and online assessments.

## Repository Structure

```text
database-leetcode/
│
├── Easy/
├── Medium/
├── Hard/
└── README.md
```

Each solution includes:

* Problem Number and Title
* SQL Query
* Approach/Explanation (where necessary)
* Time and Space Complexity (if applicable)

## Topics Covered

* SELECT Statements
* WHERE, ORDER BY, GROUP BY
* Aggregate Functions
* Joins (INNER, LEFT, RIGHT, SELF)
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* CASE Statements
* Date Functions
* Advanced SQL Concepts
* And whatever concept the platform provides
